# retrofil_example

add retrofit:
```shell
flutter pub add dio && \
flutter pub add retrofit && \
dart pub add retrofit_generator
```

add: flutter_riverpod:
```shell
flutter pub add \
flutter_riverpod \
riverpod_annotation \
dev:riverpod_generator \
dev:build_runner \
dev:custom_lint \
dev:riverpod_lint
```

add: freezed:
```shell
flutter pub add \
  freezed_annotation \
  --dev build_runner \
  --dev freezed \
  json_annotation \
  --dev json_serializable
```
